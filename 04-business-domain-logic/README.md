# 🧠 Business Domain Logic

O "cérebro" da aplicação. Aqui as regras de negócio são definidas de forma pura, sem depender de banco de dados ou frameworks.

### 🛠️ Funcionalidades:
- **Entities:** Classes que representam objetos do mundo real (User, Order, Product).
- **Service Layer:** Processamento de regras complexas (ex: cálculo de descontos, validação de fluxo de cadastro).
- **Value Objects:** Garantia de tipos de dados válidos (ex: Email, CPF, Moeda).
