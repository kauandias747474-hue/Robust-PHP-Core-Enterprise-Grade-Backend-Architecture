# 🧪 Quality & Testing Suite

Garantia de que o sistema se comporta como esperado após cada modificação.

### 🛠️ Funcionalidades:
- **Unit Tests:** Testes de funções isoladas (ex: criptografia, validadores).
- **Integration Tests:** Testes do fluxo completo (ex: salvar no banco e ler de volta).
- **Mock Objects:** Simulação de serviços externos para testes rápidos.
