# 🛠️ Utility & Helpers Toolkit

Conjunto de ferramentas reutilizáveis para acelerar o desenvolvimento e manter o código DRY (Don't Repeat Yourself).

### 🛠️ Funcionalidades:
- **String Manipulator:** Slugify, mask (CPF/CNPJ) e formatadores.
- **DateTime Tools:** Conversão de fusos horários e cálculos de períodos.
- **JSON Handler:** Wrapper seguro para `json_encode` e `json_decode` com tratamento de erros.
