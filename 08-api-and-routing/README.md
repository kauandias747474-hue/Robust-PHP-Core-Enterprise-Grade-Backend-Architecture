# 🚦 API & Routing Engine

O controlador de tráfego do sistema. Define como as URLs se conectam ao código.

### 🛠️ Funcionalidades:
- **Request Matcher:** Identifica a rota acessada e despacha para o Controller correto.
- **Middlewares:** Filtros executados antes da lógica principal (ex: Auth Check).
- **RESTful Standards:** Suporte para métodos GET, POST, PUT e DELETE.
