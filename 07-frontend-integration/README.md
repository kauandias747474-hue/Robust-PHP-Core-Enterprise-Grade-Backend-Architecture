# 🎨 Frontend Integration Layer

A ponte entre o PHP e a interface do usuário. Focada em entregar dados prontos para o consumo.

### 🛠️ Funcionalidades:
- **Response Factory:** Formatação de respostas em JSON ou HTML.
- **Asset Manager:** Gestão simples de caminhos para CSS e Vanilla JS.
- **Template Engine (Simple):** Sistema de renderização de views com escape de dados automático.
